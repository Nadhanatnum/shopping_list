## **Shopping List App**
แอปพลิเคชันจดรายการซื้อของที่พัฒนาด้วย Flutter โดดเด่นด้วยดีไซน์ Dark Minimalist และระบบอัจฉริยะที่ช่วยลดขั้นตอนการกรอกข้อมูล พร้อมบันทึกขึ้น Cloud ทันที

## **Key Features (ฟีเจอร์เด่น)**
Real-time Firebase Sync: ดึงและบันทึกข้อมูลทันทีผ่าน REST API ข้อมูลไม่สูญหายเมื่อปิดแอป

Auto-Category Selection: ระบบเดาหมวดหมู่สินค้าอัตโนมัติ (เช่น พิมพ์ "Milk" ระบบจะเลือกหมวด Dairy ให้อัตโนมัติ)

Smart Dynamic Icons: เปลี่ยนไอคอนหน้ารายการให้ตรงกับชื่อสินค้าแบบอัตโนมัติ (เช่น 🍎, 🥩, 🥛)

Modern Dark UI: ดีไซน์โทนเข้มสไตล์มินิมอล ช่องกรอกข้อมูลแบบขอบมนไร้เส้นขอบ สบายตาและทันสมัย


Form Validation & State: มีระบบตรวจจับความถูกต้องของข้อมูล และระบบ Loading ป้องกันผู้ใช้กดส่งข้อมูลซ้ำซ้อน

## **Tech Stack**
Framework: Flutter (Dart)

Backend: Firebase Realtime Database

Packages: http (สำหรับจัดการ REST API)
